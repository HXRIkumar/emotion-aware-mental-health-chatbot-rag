# Emotion-Aware Mental Health Support Chatbot using RAG and LLMs

An AI-powered chatbot designed to provide emotionally aware mental health support by combining Retrieval-Augmented Generation (RAG) with Large Language Models.

## Features

- Emotion detection from user input
- Context-aware responses using RAG
- Knowledge retrieval from mental health resources
- LLM-powered conversational support
- Scalable architecture for real-time assistance

## Tech Stack

Python  
Large Language Models (LLM)  
Retrieval-Augmented Generation (RAG)  
Transformers  
Vector Embeddings  
Google Colab  

## Architecture

1. User message input
2. Emotion detection
3. Retrieve relevant mental health knowledge
4. RAG pipeline
5. LLM generates empathetic response
